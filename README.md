# Biblioteca en C#

Este proyecto es una aplicación de consola en C# que simula una biblioteca. Permite gestionar libros y usuarios, así como realizar préstamos y devoluciones.

## 📌 Funcionalidades

- Gestión de libros: agregar, eliminar y buscar libros.
- Gestión de usuarios: agregar, eliminar y buscar usuarios.
- Préstamos: realizar y registrar préstamos de libros a usuarios.
- Devoluciones: registrar la devolución de libros prestados.

## 🛠️ Tecnologías Utilizadas

- C#
- .NET Framework
- Programación Orientada a Objetos (POO)
